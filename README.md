# Health Gen-AI

## Project Setup

To set up the Health Gen-AI project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/malc3om/Health_Gen-Ai.git
   cd Health_Gen-Ai
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```



For detailed documentation, refer to the [Wiki](https://github.com/malc3om/Health_Gen-Ai/wiki).
